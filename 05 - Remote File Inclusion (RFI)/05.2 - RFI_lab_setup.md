> Clonamos el repositorio que nos servirá de laboratorio

```
git clone https://github.com/vavkamil/dvwp.git
```

> Nos movemos de directorio

```
cd dvwp
```

> Levantamos el laboratorio

```
docker-compose up --build -d
```

> Accedemos a wordpress y nos creamos una cuenta

```
firefox "http://localhost:31337"
```

> Instalamos algunos plugins vulnerables

```
docker-compose run --rm wp-cli install-wp
```

> Buscamos alguna version de algún plugin vulnerable a RFI mediante searchploit como gwolle

> Modificamos los permisos de la carpeta wp-content y agregamos una directiva al archivo php.ini

```
docker exec -it $(docker ps -aq | tail -n 2 | head -n 1) bash
```
```
root@wordpress-id:/var/www/html# chown www-data:www-data -R wp-content/
```
```
root@wordpress-id:/var/www/html# echo -e "display_errors = \"on\" \nallow_url_include = \"on\"" > php.ini
```
