Montamos un servidor http con python para simular un entorno de pre-producción

```
python3 -m http.server 4646
```

Probamos el acceso al login de pre-producción

<img width="1171" height="744" alt="image" src="https://github.com/user-attachments/assets/0c4057e7-af96-425f-bfd8-2465c6585435" /><br>

Agregamos el parámetro --bind al servidor http con python para que sea solo accesible desde el propio contenedor

```
python3 -m http.server 4646 --bind 127.0.0.1
```
<img width="811" height="95" alt="image" src="https://github.com/user-attachments/assets/bff8d5db-a368-4e14-9893-d05fcce94346" /><br>
