Limpiamos contenedores anteriores
```
docker stop $(docker ps -aq) && docker rm $(docker ps -aq)
```
Creamos una nueva red
```
docker network create --driver=bridge network1 --subnet=10.10.0.0/24
```
Creamos la máquina (producción) para la red interna del laboratorio
```
docker run -dit --name PRO ubuntu && docker network connect network1 PRO
docker exec PRO bash -c "apt update && apt install -y iproute2 net-tools iputils-ping"
```
Creamos la máquina (pre-producción) para la red interna del laboratorio
```
docker run -dit --name PRE --network=network1 ubuntu
docker exec PRE bash -c "apt update && apt install -y iproute2 net-tools iputils-ping"
```
Creamos la máqunia (atacante) del laboratorio
```
docker run -dit --name ATTACKER ubuntu
docker exec ATTACKER bash -c "apt update && apt install -y iproute2 net-tools iputils-ping"
```
